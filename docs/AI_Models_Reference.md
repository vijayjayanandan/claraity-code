## Model Categories

### Text Generation Models
- **Claude Series**: Balanced performance models (Haiku for speed, Sonnet for quality)
- **GPT Series**: OpenAI's flagship models (GPT-4o, GPT-5 variants)
- **Gemini Series**: Google's multimodal models
- **Mistral Series**: Open-source high-performance models
- **Llama**: Meta's open-source models

### Code-Specific Models
- **Cursor Series**: Specialized for code editing and generation
- **Qwen3-coder-next**: Advanced code generation

### Image Generation
- **DALL-E 3**: Text-to-image generation
- **Imagen Series**: Google's image generation models

### Embedding Models
- **text-embedding-3-large/small**: OpenAI embeddings
- **text-embedding-ada-002**: Classic embedding model
- **kaim-embedding-gemini3-12b**: Gemini-based embeddings
- **qwen3-embedding-6b**: Qwen embeddings

### Audio Models
- **Whisper-1**: Speech-to-text transcription
- **TTS-1/TTS-1-HD**: Text-to-speech synthesis
- **GPT-4o-transcribe**: Advanced audio transcription

### Specialized Models
- **DeepSeek-OCR**: Optical character recognition
- **TranslateGemma-27b**: Translation tasks
- **GPT-4o-content-filter**: Content moderation
